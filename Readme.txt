Создать виртуальную машину в google cloud.
Задание:
1) Создать виртуальную машину (cервер №1 - ubuntu)
2) Установить apache http сервер
3) Написать CGI скрипт на shell
4) При http GET запросе на сервер с клиентского устройства данный скрипт должен отдать какой-либо бинарный файл системы, указанный в параметре.
            Пример:
                        wget http://<server-ip>/getFile?path=/bin/bash
            Результат:
                        wget http://<server-ip>/getFile?path=/bin/bash
                        --2020-10-28 17:41:42--  http://<server-ip>/getFile?path=/bin/bash
                        Connecting to <server-ip>:80... connected.
                        HTTP request sent, awaiting response... 200 OK
                        Length: unspecified [text/plain]
                        Saving to: ‘bash’
                        bash                                                       [ <=>   ]     394  --.-KB/s    in 0.1s    
                        2020-10-28 17:41:43 (2.97 KB/s) - ‘bash’ saved [394]
 
            В данном случае параметр path=/bin/bash означает, что требуется передать клиенту файл, расположенный по пути /bin/bash
            В результате выполнения такого запроса, клиент скачает с сервера №1 файл, расположенный по указанному пути.
* 5) Создать еще одну виртуальную машину (сервер №2), к которой будет переадресовываться запросы с сервера №1
            При получении запроса на скачивание файла сервером №1, CGI скрипт, посредством ssh, получает файл с сервера №2.
